# 🧟 Zombie Shooter

A terminal-based zombie shooter game written in C. Fight waves of zombies in single-player or co-op mode!

## 🎮 Features

- **Single-player mode** (WASD + Space)
- **2-Player co-op mode** (P1: WASD+Space, P2: IJKL+J)
- **Auto-saving high scores**
- **Terminal-based graphics using ncurses**
